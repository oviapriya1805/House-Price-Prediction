**Project Name:** House Price Prediction using Machine Learning

**Project Scope:**
This project predicts house prices using Machine Learning techniques. The California Housing Dataset is preprocessed and analyzed to identify the factors affecting house prices. A Linear Regression model is trained on the dataset to predict house prices based on features such as median income, house age, average rooms, average bedrooms, population, average occupancy, latitude, and longitude. The trained model provides accurate price predictions and is evaluated using standard performance metrics.

### Result

The Linear Regression model successfully predicts house prices based on housing features with good accuracy. The model was evaluated using MAE, MSE, and R² Score, demonstrating its effectiveness in estimating house prices. The trained model was saved for future predictions and can be integrated into a web application for real-time house price estimation.
